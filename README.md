# 👋 환전사이트 프로젝트 입니다!

<br>


## 👭 프로젝트 요약


코로나가 풀리면서 해외여행에 관심이 증가하는 만큼, 환율정보에 대한 수요도 증가하고 있어 <br> 간편하게 환율 조회를 할 수 있는 사이트를 구현했습니다. 

- 개발 기간: 2022.09,17 - 2022.10.02
- 팀 프로젝트(2인)

| 분류 | 강지연 | 성신혜 |
| --- | --- | --- |
| 기능구현 | 유저 관리 <br> 인터셉터, 환율 정보 계산, sns 회원가입  | 게시판 관리 <br> 통화별 환율 조회, 환율알림서비스, sns 회원가입 |
| 프론트 | 기본 틀 구성, css | 기본 틀 구성, css |
| API | 결제, 공공데이터 포털 환율정보 조회, naver login api | kg 이니시스 본인인증, 공공데이터 포털 환율정보 조회  |
| 기타 | - | 서버 aws, 기획 |
